## Next.js 项目搭建计划

### 1. 环境检查与准备
- 检查 Node.js 版本（需要 18.17 或更高版本）
- 检查 npm/yarn/pnpm 包管理器
- 确认 Git 已安装并配置

### 2. 创建 Next.js 项目
- 使用 `create-next-app` 创建新项目
- 配置项目选项：
  - TypeScript: 启用
  - ESLint: 启用
  - Tailwind CSS: 启用
  - src/ 目录: 启用
  - App Router: 启用
  - 导入别名: 启用 (@/*)

### 3. 项目基础配置
- 配置 TypeScript 严格模式
- 调整 ESLint 规则
- 配置 Prettier 代码格式化
- 设置 Git 仓库和 .gitignore

### 4. 安装额外依赖
- 安装 Framer Motion（动画库）
- 安装其他必要工具（如：husky、lint-staged）

### 5. 项目结构初始化
- 创建基础文件夹结构
- 设置全局样式和主题
- 创建基础组件和布局

### 6. 开发环境验证
- 启动开发服务器
- 确认所有配置正常工作
- 进行基本的功能测试

这个计划将在当前目录下创建一个符合品牌官网开发需求的 Next.js 项目基础环境。
